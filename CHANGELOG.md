# 更新日志

所有重要的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-XX-XX

### ✨ 新增

- 🎼 可视化乐谱编排功能
- 🎹 完整的调性选择系统（12音名 + 大小调 + 中古调式 + Blues）
- 🎸 25+ 种乐器配置，支持搜索和自定义添加
- 🌓 深色/浅色双主题切换
- 📊 PDF/PNG/JSON 三种导出格式
- 🔄 JSON 数据导入导出功能
- 📖 首次使用教程
- 🎯 段落长度按小节数比例显示
- 🎨 现代化的 UI 设计

### 🎸 乐器支持

- 人声：男主唱、女主唱、和声
- 吉他：主音吉他、节奏吉他、木吉他
- 贝斯：贝斯、低音提琴
- 鼓组：鼓、打击乐
- 键盘：键盘、钢琴、合成器、管风琴
- 弦乐：弦乐组、小提琴、大提琴
- 管乐：萨克斯、小号、长号、长笛、单簧管、口琴
- 电子：DJ/搓盘、采样器

### 📦 技术特性

- 纯前端实现，无需后端
- 响应式设计
- CSS Variables 主题系统
- Drag & Drop API
- LocalStorage 数据持久化
